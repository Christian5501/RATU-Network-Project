# RATU Network Project

Design and implementation of an AI-enabled Zero Trust secure multi-campus 
university network for Rwanda Advanced Technology University (RATU), 
covering three campuses: Kigali, Musanze, and Huye.

This project demonstrates enterprise network design, VLAN segmentation, 
OSPF routing, Zero Trust security architecture, site-to-site VPN, 
Python network automation, and hybrid cloud integration concepts.

**Status:** In progress
**Tools:** Cisco Packet Tracer, Python, GitHub
